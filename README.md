# 超景气价值投机 · 选股助手

> 基于公开分享的"超景气价值投机"投资体系，构建一个会话式的选股与投研助手。

## 项目初心

某位投资者在 2024-2025 年系统性地录制了十期《超景气价值投机》教学视频，沉淀出一套从事件识别、板块判断、个股筛选到止盈管理的完整框架。这套框架的长处在于**认知逻辑清晰**，短板在于**落地时高度依赖人工刷新闻、查财报、翻数据**。

本项目的目标：**把这套认知框架固化为一个 AI Skill**，让每一次选股判断都跑在同一套标准上。

**不做的事**：
- ❌ 不做量化策略，不做回测，不做实盘交易
- ❌ 不做自动扫描、定时任务、实时监控推送
- ❌ 不替代人工决策——最终买卖仍由人拍板

**真实用法**：你有闲钱 / 在做月度投资计划 / 看到一个新主题时，打开 WorkBuddy 会话，扔一个股票或主题进来，AI 按体系走一遍给结论。低频、深度、一次性。

## 体系核心（一句话版）

> 在超景气行业的拐点前夜布局，通过"现象级事件 / 困境反转 → 超景气板块 → 纯度筛选 → 信息整理 + 四档定档"四步定位标的，赚企业基本面与逻辑发生重估的钱。

**两类选股入口**（2026 更新版）：
1. **现象级事件** — 科技突破/政策驱动/地缘冲突/海外映射，需满足真实性/传播性/规模性/时效性四要素
2. **困境反转** — 行业已经惨到极致后的逆向布局，适用政策强制干预（限产/环评）或市场出清；**白酒房地产不适用**

详细体系见 [`超景气价值投机体系总结.md`](./超景气价值投机体系总结.md)。

## Skill 工作流

Skill 提供两个核心入口：

| 入口 | 作用 |
|---|---|
| `/score <股票代码>` | 对单只股票走完整工作流，给出"四档定性标签 + 详细信息" |
| `/analyze-theme <主题>` | 给定产业主题/催化剂，做产业链拆解 + 候选股池 |

### 输出形态

**不打数字分**（避免假精确），改为**双层结构**：

**顶层（5 秒看懂）**：四档标签 + 一句话理由 + 预警旗帜
- 🟢 **重点关注**：三门槛过 + 笨韭双击成立 + 估值合理
- 🟡 **可打野**：基本面好但时点/估值不理想
- ⚪ **观察等待**：信号缺失，暂不动手
- 🔴 **不予配置**：门槛未过 / 命中一票否决

**底层**：按七维结构展开的详细信息（数据来源、计算假设、不确定性提示），用户可下钻审视。

### 七维结构

| 维度 | 角色 | 数据源 |
|---|---|---|
| 1. 行业景气度 | 🚪 门槛 + 入口 | Web Search（政策密度 + 现象级事件 + 用户输入）|
| 2. 业务纯度 | 🚪 门槛 + 加权 | akshare `stock_zyjg_em` |
| 3. **动态预测估值** | 📊 信息整理（不打分） | akshare 财报 + `web_fetch` 抓年报"经营计划"段落 |
| 4. 龙头判断 | 📊 信息整理 | **市占率 + 技术代差 + 代际卡位**（不看营收/市值排名） |
| 5. 市场辨识度 | 提示项 | 人气 + 名字辨识 + 历史标签 + 稀缺性（**作者明说权重不高**） |
| 6. 个股风险值 | 🚪 门槛（一票否决） | akshare 公告 + 反向证伪 Web Search |
| 7. 流动性环境 | 🎚️ 市场环境 | akshare 两市成交额 |

⭐ **关键判别**：在七维之外独立跑"**笨韭双击识别**"——行业拐点 + 企业基本面拐点是否同时出现，这是判 🟢 还是 🟡 的核心依据。

## 数据路径

| 工具 | 用途 | 调用方式 |
|---|---|---|
| **akshare**（首选） | 主营构成、财务、股价、公告 | 在 AI 会话内由 AI 写 Python 调用 |
| **web_fetch**（必要） | 年报"经营计划/未来展望"段落抓取 | 在 AI 会话内由 AI 调用 |
| **Web Search**（兜底 + 反向证伪） | 产业情报、市占率、风险扫描 | 在 AI 会话内由 AI 调用 |
| 人工输入 | 现象级事件信号 | 由用户提供 |

**前置依赖**：本机已安装 Python 3.10-3.12 64 位 + akshare（`py -m pip install akshare`）。

**架构说明**：本项目不做脚本化、不做 MCP 外部调用、不做定时任务。所有数据查询都在 AI 会话内完成。

## 打分模型的工程化立场

原作者在视频中明确表示自己"平时很少用打分模板，比值是主观经验"。本项目**不硬复刻数值公式**，改为：

1. **不打数字分**：所有维度只输出"信息整理 + 假设清单"，不输出 73 分这种假精确
2. **AI 不替用户决策**：最终标签是"体系视角下的当前状态"，不是"建议买入"
3. **关键假设全部明示**：单价、净利率、业务结构、同业对标的可比性——一切让用户能质疑
4. **半机械化定档**：主流程严格按规则走，AI 允许上调/下调一档但必须给理由

## 使用对象

本项目面向个人投资者，覆盖 A 股 + 港股 + 美股，遵循"超景气价值投机"流派（非"价值投资吃息"派）。使用者需自行判断体系是否契合个人风险偏好。
