## 四档标签判据

### 🟢 重点关注

**必须同时满足**：
1. 三门槛全过（行业景气度 ≠ 0、业务纯度足够、风险无一票否决）
2. **双拐点共振成立**（行业拐点 + 企业基本面拐点同时出现）
3. 预计估值在合理或便宜区间（PE < 100 / PS < 20）

**典型形态**：
- 行业刚出现现象级事件（< 6 个月内）
- 企业近 4 季度净利从亏损转盈或加速
- 主营业务 60%+ 在景气赛道
- 当前估值还没把未来业绩打满

**示例**：思特威（2023 年）、速腾（2025 Q4）

### 🟡 可打野

**满足任一**：
1. 三门槛全过 + **行业单拐点**（只有行业拐点，企业基本面尚未确认）
2. 三门槛全过 + 已是明牌龙头但估值偏贵（市场已充分定价）
3. 三门槛全过 + 拐点已半兑现（涨幅较大但故事还没走完）

**典型形态**：
- 龙头地位明确但 PE > 100 / PS > 20
- 行业景气但企业业绩还没拐头
- 题材已被市场广泛认知

**示例**：禾赛（如果剥离地缘风险）、应流（2026 当前）

### ⚪ 观察等待

**满足任一**：
1. 三门槛勉强过但关键信号缺失
2. 年报指引不明，无法做动态估值
3. 业绩拐点未确认（数据期数不够）
4. 催化薄弱（没有现象级事件，只有基本盘景气）

**典型形态**：
- 行业刚出现催化但还没扩散
- 公司基本面拐点信号弱
- 数据缺失严重（如新上市不足 2 年）

### 🔴 不予配置

**满足任一即触发**：
1. 行业景气度 = 0（行业明显下行 / 体系排除的赛道）
2. 业务纯度过低（目标景气赛道 < 30% 营收）
3. 风险一票否决（实控人重大减持 / 财务造假 / 监管立案 / 重大诉讼 / 公司治理崩塌 / 地缘政治硬制裁）

**典型反例**：
- 金种子酒（白酒下行 + 体系排除）
- 中国中免（行业景气度 0）
- 南都电源（管理层套现 + 重组跳票）
- 禾赛（地缘政治硬制裁 + 兑现承诺差）

## AI 微调机制（半机械化关键）

允许 AI 主动**上调或下调一档**，但必须：

### 上调规则

- 必须有强证据（不能凭直觉）
- 不能跨档（🔴 不能直接调到 🟢）
- 必须把"为什么上调"明确写出来

**典型上调场景**：
- 边缘情况下双拐点共振信号特别强（毛利率 + 营收 + 净利同时拐头）
- 估值偏贵但同业对标显示行业整体被低估
- 龙头地位有代际卡位加分（如下一代技术领先）

### 下调规则

- 上调要严，下调可宽
- 重大软风险但不到一票否决，可以下调一档

**典型下调场景**：
- 公司治理担忧（管理层套现没到一票否决但值得警惕）
- 估值严重透支（PE > 100 但还未到 200）
- 流动性环境危险（市场成交额接近 4 万亿）

## 预警旗帜清单

不影响标签判定，但必须显示在标签下方：

| 旗帜 | 触发条件 |
|---|---|
| ⚠️ 大股东减持 | 已披露的减持计划 / 控股股东减持转债 / 高管减持（未到一票否决程度） |
| ⚠️ 估值透支 | PE > 100 / PS > 20 但仍在 🟡 档 |
| ⚠️ 公司治理担忧 | 管理层动荡、套现、重组跳票（未崩塌但值得警惕） |
| ⚠️ 地缘政治风险 | 关税、清单、出口管制相关 |
| ⚠️ 承诺未兑现 | 产品延期、目标未达、业绩预告下修 |
| ⚠️ 流动性环境危险 | 当前两市成交额 ≥ 3.5 万亿 或 ≤ 2 万亿 |
| ⚠️ 年报指引不明 | 无法做动态估值 |
| ⚠️ 数据缺失严重 | 新股 / 内置股票数据 接口报错 / 关键数据拉不到 |

## 七维详细规则速查

### 维度 1：行业景气度（门槛）

**判 0 分**：
- 白酒、房地产（2024-2026 体系排除）
- 行业整体亏损扩大、需求长期下行
- 无任何政策催化、技术突破、海外映射

**判 40-100**（仅作 AI 内部参考，不影响门槛）：
- 政策密度（过去 3-6 月）
- 现象级事件强度
- 三击验证（政策 + 技术 + 需求 几击）

**不看**：板块涨幅、资金流向（违背"跑在前面"原则）

### 维度 2：业务纯度（门槛 + 加权）

**用 内置股票数据 `stock_zyjg_em` 精确取数**

**门槛判 pass**：
- 目标景气赛道占营收 < 30%
- 业务横跨 5+ 个不相关赛道（典型蹭概念）

**加权打分**（参考）：
- 70% 以上 → 高分
- 50-70% → 中分
- 30-50% → 低分

### 维度 3：动态预测估值（信息整理，不打分）

**4 步子流程**详见 SKILL.md Step 5。

**关键判据**：
- PE < 25 / PS < 5 → 便宜
- PE 25-50 / PS 5-10 → 合理
- PE 50-100 / PS 10-20 → 偏贵但可接受
- PE > 100 / PS > 20 → 严重透支
- 算不出来 → ⚠️ 让用户介入

**所有假设必须明示**：单价、净利率、业务结构、同业对标的可比性

### 维度 4：龙头判断

**只看三件事**：
- 市占率（口径要明示）
- 技术代差（是否在下一代技术领先）
- 市占率斜率（过去 3 年变化）

**档位**：
- 全球第一 → 龙头满分
- 全球前三 + 代际领先 → 高分
- 国内细分龙头（市占率国内第一但海外被压制） → 中高分
- 跟随者 → 低分
- 边缘 → 低分

**加分项**：
- 代际卡位（增量市场先发） +
- 市占率斜率上升（如天岳 12%→27.6%） +

⚠️ **不看营收/市值排名**——那是题材龙头。

### 维度 5：市场辨识度（提示项）

**体系口径认为"权重不高"**。

四要素任一拉满即高分：
- 人气度
- 名字辨识
- 历史操盘标签
- 稀缺性

⚠️ **不要因为这一维单独上调标签**。它只是提示项。

⚠️ 辨识度 ≠ 预期差。辨识度是"资金会不会优先想到它"，预期差是"市场还没反应过来"。两者方向相反。

### 维度 6：个股风险值（一票否决 + 预警）

**数据来源**：内置股票数据 公告 + 反向证伪 Web Search。

**一票否决**：
- 实控人重大减持（>5% 股本）
- 控股股东大额减持（含可转债，>10%；但父传子等平稳传承需降级处理）
- 财务造假 / 监管立案 / 重大诉讼悬顶
- 公司治理崩塌（管理层动荡 + 套现 + 重组跳票）
- 地缘政治硬制裁（实体清单 / 1260H 清单 / 出口管制）

**预警但不直接否决**：
- 已充分披露且市场已消化的减持计划
- 小额行政处罚
- 一般诉讼或专利纠纷（金额不构成重大影响）
- 软性风险：承诺未兑现、PR 过度、管理层信誉弱

### 维度 7：流动性环境

通过 内置股票数据 取当前 A 股两市成交额：

| 成交额 | 状态 | 系数 |
|---|---|---|
| < 2 万亿 | 冰点 | 0.7 |
| 2-2.5 万亿 | 冷淡 | 0.9 |
| 2.5-3 万亿 | 常态 | 1.0 |
| 3-3.5 万亿 | 活跃 | 1.1 |
| 3.5-4 万亿 | 危险 | 0.8 |
| ≥ 4 万亿 | 极度危险 | 0.7 |

**不影响个股标签**，只在最终结论里加一句"当前市场流动性 X 档"。

## 双拐点共振 / 单击 判别速查

### 双击 = 行业拐点 + 企业基本面拐点 同时出现

**行业拐点判定**：
| 信号类型 | 示例 |
|---|---|
| 现象级事件触发 | 华为 P60 → CIS 景气 / 智能眼镜爆发 / AI 数据中心缺电 |
| 政策强力催化 | 商业航天 / 低空经济 / 固态电池 / 反内卷 |
| 海外映射 | 海外龙头股价拉升 → 国内对标 |
| 渗透率 0→1 进入 1→10 | 智能驾驶 L3 / 人形机器人 |

**企业基本面拐点判定**（用 内置股票数据 4-5 年净利曲线）：
| 信号 | 判据 |
|---|---|
| V 型反转 | 连续亏损 → 转盈 → 加速（如思特威 -0.83→0.14→3.93→10）|
| 毛利率拐头 | 从负转正（如速腾 ADAS -5.9%→+13.4%）|
| 营收加速 | 连续两年增速上升 |
| 单季度首次盈利 | 季报里首次扭亏 |

### 三档结论

- **🟢 双击成立**：行业 + 企业 都拐头 → 体系核心机会
- **🟡 单击成立**：只有行业拐点（企业基本面尚未确认） → 仍可关注，但仓位小
- **⚪ 无信号**：行业景气但企业未受益 / 企业改善但行业逆风 → 不进重仓

## 反向证伪 Web Search 模板

每只股必跑（防止只拉到正面信息）：

```
1. "{公司名} 风险 / 诉讼 / 监管 / 立案"
2. "{公司名} 承诺 / 延期 / 兑现 / 跳票"
3. "{公司名} 减持 / 实控人变更 / 高管"
4. "{公司名} 2026 业绩 / 预告"
5. "{公司名} 质疑 / 投诉 / 投毒"  （针对喂屎型公司）
```

⚠️ 不能只搜利好关键词。必须正反双搜。

## 数据源污染识别

| 信息源 | 可信度 | 用法 |
|---|---|---|
| 公司年报 / 季报 | 🟢 高 | 直接采信 |
| 上交所 / 深交所公告 | 🟢 高 | 直接采信 |
| 港交所披露易 / SEC EDGAR | 🟢 高 | 直接采信 |
| 内置股票数据 财务数据 | 🟢 高 | 直接采信 |
| 第三方权威机构（Yole / TSR / TrendForce） | 🟡 中 | 大体可信但有付费报告偏向 |
| 卖方研报 | 🟡 中 | 立场已知，结论打折 |
| 财经媒体（财联社 / 上海证券报） | 🟡 中 | 转述类内容相对可信 |
| 自媒体 / 公众号 | 🔴 低 | 仅作线索，不当证据 |
| 公司 PR 稿 | 🔴 低 | **特征：自媒体大量转载、措辞夸张、缺少法律责任披露** |

⚠️ **重要原则**：
- 利好信息要交叉验证（PR 稿 vs 年报 vs 第三方报告）
- 利空信息相对可信（没人会主动黑自己）
- "全球第一"、"全网最强"、"颠覆性"等词汇出现 → 警惕 PR 稿
- 同一信息只在自媒体出现、年报里没披露 → 不当独立证据
