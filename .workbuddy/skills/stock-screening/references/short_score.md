# 短线七维共振标签参考

> 这是 stock-screening 的短线辅助参考。当用户需要从短线技术面角度辅助选股时（1-5 天博弈），可在 /score 之外叠加此参考。
> 本参考不替代主体系的四档标签判定，但使用相同的 🟢🟡⚪🔴 标签体系，仅作为短线操作视角的补充。

---

## 标签体系

| 标签 | 含义 | 短线策略 |
|---|---|---|
| 🟢 **短线强势** | 多维度共振偏多，主力资金配合，短线信号明确 | 可积极参与，按操作计划执行 |
| 🟡 **短线可参与** | 部分维度偏多，但存在矛盾信号或资金中性 | 可轻仓参与，注意节奏，严格止损 |
| ⚪ **短线观望** | 多空交织、信号不明确，或处于变盘窗口 | 持仓观望或空仓等待方向确认 |
| 🔴 **短线回避** | 多数维度偏空，或主力资金明显出逃/对倒诱多 | 不参与，持有者逢高减仓或离场 |

---

## 指标体系总览

基于"均线 + MACD + KDJ + RSI + 布林带 + 量价 + 主力资金"七维共振体系，逐维出具信号（🟢偏多 / 🔴偏空 / ⚪中性），综合判定短线标签。

| 维度 | 名称 | 权重等级 |
|---|---|---|
| 1 | 趋势与支撑阻力（均线 + 布林带） | 基础 |
| 2a | MACD | 中 |
| 2b | KDJ | 中 |
| 2c | RSI | 中高（背离优先） |
| 3 | 量价关系 | 中 |
| 4 | 主力资金流向 | **最高** |
| 5 | K 线形态 | 辅助 |
| 6 | 市场环境 | 背景调节 |

## 数据拉取

需要的数据及命令（详见 `data_sources.md`）：

| 数据类型 | 命令 |
|---|---|
| 实时行情快照 | `quote <symbol>` |
| K 线数据（日线，≥ 200 根） | `kline <symbol> --period day --limit 200` |
| 技术指标全量 | `technical <symbol> --group all` |
| 资金流向（A 股） | `asfund <symbol>` |
| 资金流向（港股） | `hkfund <symbol>` |
| 市场环境 | `quote sh000001` + `quote sz399001` |

**`technical --group all` 字段映射**：

| 指标 | 字段 |
|---|---|
| 均线 | `MA5`, `MA10`, `MA20`, `MA60` |
| MACD | `DIFF`, `DEA`, `MACD`（柱） |
| KDJ | `K`, `D`, `J` |
| RSI | `RSI1`（6日）, `RSI2`（12日）, `RSI3`（24日） |
| 布林带 | `BOLL_UP`（上轨）, `BOLL_MID`（中轨/20日）, `BOLL_LOWER`（下轨） |

## 维度 1：趋势与支撑阻力（均线 + 布林带）

| 判定 | 信号 | 条件 |
|---|---|---|
| 多头强化 | 🟢 | 5日 > 10日 > 20日，股价在 20 日线上方；布林开口向上，价格沿上轨 |
| 空头强化 | 🔴 | 5日 < 10日 < 20日，股价在 20 日线下方；布林开口向下，价格沿下轨 |
| 变盘预警 | ⚪ | 均线粘合交织；或布林带极度收口（带宽缩至近期 20 日最窄）。⚠️ 标注预警 |

**判据要点**：
- 只看 MA5、MA10、MA20 三线排列，MA60 做远期参考
- 布林带宽缩至近期最窄水平 → 变盘预警，方向须结合 D4 资金流向判断

---

## 维度 2a：MACD

| 场景 | 信号 |
|---|---|
| 零轴上 + 二次金叉 or 底背离 + 红柱放大 | 🟢 强多 |
| 零轴下 + 二次金叉（弱势反转） | 🟢 弱多 |
| 零轴附近粘合、无明确方向 | ⚪ 中性 |
| 零轴上 + 死叉（强势回调） | 🔴 弱空 |
| 零轴下 + 死叉 or 顶背离 + 绿柱拉长 | 🔴 强空 |

**判据要点**：
- "二次金叉"：近期 20 日内已有一次金叉，正在再次金叉
- "底背离"：股价创阶段新低，DIFF 值不跟随新低
- "顶背离"：股价创阶段新高，DIFF 值不跟随新高
- 背离信号覆盖常规金叉/死叉，不叠加

---

## 维度 2b：KDJ

| 场景 | 信号 |
|---|---|
| K/D 在 20 以下 + 金叉向上 | 🟢 超卖反弹 |
| K/D 在 80 以上 + 死叉向下 | 🔴 超买回调 |
| K/D 在 20-80 区间，或钝化 | ⚪ 中性 |

**判据要点**：
- K 值 > 80 且高位钝化（连续 3 日不回落）→ 标注强势延续，判 ⚪
- K 值 < 20 且低位钝化 → 标注弱势延续，判 ⚪
- J 值辅助：J > 100 = 极度超买；J < 0 = 极度超卖

---

## 维度 2c：RSI

| 场景 | 信号 |
|---|---|
| RSI < 30 + 底背离 | 🟢 强多（极高胜率反转） |
| RSI < 30（仅超卖，无背离） | 🟢 弱多 |
| RSI 30-70 | ⚪ 中性 |
| RSI > 70（仅超买，无背离） | 🔴 弱空 |
| RSI > 70 + 顶背离 | 🔴 强空（极高胜率见顶） |

**判据要点**：
- 底背离：股价创新低，RSI 不创新低；顶背离：股价创新高，RSI 不创新高
- 以 RSI1（6日）/ RSI2（12日）为准
- RSI 背离在三子维度中可信度最高

---

## 维度 3：量价关系

| 场景 | 信号 | 条件 |
|---|---|---|
| 健康上涨 | 🟢 | 涨幅 > 1% 且成交量较 5 日均量放大 ≥ 20% |
| 缩量企稳 | 🟢 | 价平量缩，连续缩量至 60 日均量 50% 以下，预示变盘 |
| 中性 | ⚪ | 无显著异常 |
| 放量下跌 | 🔴 | 跌幅 > 1% 且成交量较 5 日均量放大 ≥ 30% |
| 量价背离（诱多） | 🔴 强空 | 价升量缩（涨 > 2% 但量缩 > 10%），或放量滞涨（量放大 > 30% 但涨 < 0.5%） |

**判据要点**：
- 与 D4 资金流向交叉验证：价升量增 + 主力流入 → 确认；价升量增 + 主力流出 → 降级
- 缩量企稳：在 20 日线上方 = 加仓信号；在 20 日线下方 = 暂缓跌势

---

## 维度 4：主力资金流向（最高权重）

| 场景 | 信号 | 条件 |
|---|---|---|
| 真实拉升 | 🟢 强多 | 股价 ↑ + 超大单/大单连续净流入（3 日≥ 2 日净流入）+ 筹码低位密集 |
| 恐慌吸筹 | 🟢 弱多 | 股价 ↓ + 主力小幅流入 + 成交量萎缩（洗盘特征） |
| 中性 | ⚪ | 无显著单边流向，或数据缺失 |
| 明确派发 | 🔴 强空 | 股价横盘/微涨 + 主力连续净流出 + 成交量放大 |
| 对倒诱多 | 🔴 强空 | 股价拉升 + 主力净流出；中单/小单大量卖出（假突破真出货） |

**判据要点**：
- 占流通市值比例 > 0.1% → 信号更可靠
- 超大单偏向机构/游资，大单可能含游资拆单
- ⚠️ **技术面再好看 + D4 对倒诱多 / 明确派发 = 一票否决为 🔴 短线回避**

## 维度 5：K 线形态

| 场景 | 信号 | 形态 |
|---|---|---|
| 强势突破 | 🟢 | 大阳线（涨幅 > 3%）突破关键均线或平台压力，放量配合 |
| 看涨形态 | 🟢 | 锤子线、启明星、上升三法、多方炮、早晨之星 |
| 十字星/纺锤 | ⚪ | 多空均衡，⚠️ 注意次日方向 |
| 无明显特征 | ⚪ | |
| 看跌形态 | 🔴 | 长上影（高位）、吊颈线（高位）、黄昏之星、三只乌鸦、阴跌 |
| 破位下跌 | 🔴 | 大阴线（跌幅 > 3%）跌破关键支撑，放量配合 |

## 维度 6：市场环境

| 两市成交额 | 状态 | 影响 |
|---|---|---|
| < 2 万亿 | 冰点 | 短线胜率降低，优先 ⚪ 或 🔴 |
| 2-2.5 万亿 | 冷淡 | 需精选，多空信号打折扣 |
| 2.5-3 万亿 | 常态 | 正常判定 |
| 3-3.5 万亿 | 活跃 | 短线友好，🟢 信号可信度上升 |
| 3.5-4 万亿 | 危险 | 过热预警，警惕变盘，偏谨慎 |
| ≥ 4 万亿 | 极度危险 | 高波动，优先 ⚪ 或 🔴 |

无法获取成交额时 → 标 ⚠️ 数据缺失，按常态处理。

## 标签判定规则（半机械化 + AI 微调）

### 主流程

逐维出具信号后，按以下优先级综合判定：

1. **一票否决**：D4 为 🔴 强空（对倒诱多 / 明确派发）→ 直接给 **🔴 短线回避**
2. **强多共振**：D4 🟢 + MACD/RSI 中至少一项 🟢 强多 + D1 🟢 → **🟢 短线强势**
3. **偏多格局**：D4 🟢 或 ⚪，且 D1+D2a+D2c 中 ≥ 2 项 🟢 → **🟡 短线可参与**
4. **偏空格局**：D1 🔴 + D2a/D2c 中 ≥ 1 项 🔴 + D3 🔴 → **🔴 短线回避**
5. **信号矛盾**：多空信号各半，D4 ⚪ → **⚪ 短线观望**
6. **变盘窗口**：D1 变盘预警，其余维度信号弱 → **⚪ 短线观望**
7. **市场环境调节**：D6 冰点/极度危险时，🟡 降为 ⚪，🟢 降为 🟡

### AI 微调机制

允许 AI 上调或下调一档，但必须：
- 写明理由
- 不能跨档（🟢 不能直接调到 🔴）
- D6 活跃环境可上调，冰点/危险环境可下调

---

## 特殊情形处理

### 变盘预警

D1 触发变盘预警时：标签至少 ⚪，显式标注 ⚠️ "均线粘合 / 布林极度收口，近期将选择方向"，结合 D4 资金流向预判方向。

### 背离与常规信号叠加

同一子维度同时出现背离和常规信号时，**只取背离信号**。MACD/KDJ/RSI 各自独立判定。

### 资金流向与技术面矛盾

当 D4 与技术维度方向相反时，**以 D4 为准**。报告须明确点出："技术面呈现 [XX]，但主力资金 [流入/流出]，故判定为 [诱多/洗盘]"。

### 数据缺失

- 数据缺失的维度 → ⚪ 中性，标注 ⚠️
- D4（资金）缺失 → 标签最多给到 🟡，且标注 ⚠️ "缺少最高权重资金验证"
